# Equity Momentum Scanner

## Overview
The Equity Momentum Scanner allows users to analyze stock momentum and identify potential investment opportunities. 

## Deployment Instructions for Streamlit Cloud

### Step 1: Connect your GitHub account
1. Go to [Streamlit Cloud](https://streamlit.io/cloud).
2. Click on 'Sign in' and choose 'Sign in with GitHub'.
3. Authorize Streamlit to access your GitHub account.

### Step 2: Deploy the app
1. After signing in, click on 'New App'.
2. Choose the repository `kranthistk/Equity-Momentum-Scanner` from the dropdown menu.
3. Select the branch (typically 'main') and the main file (e.g., `app.py`).
4. Click 'Deploy' to launch your app.

### Step 3: Generate the URL
- Your app will be assigned a unique URL. You can find this in your Streamlit Cloud dashboard. It will typically look like: `https://your-app-name.streamlit.app`.

## Features Overview
- Real-time stock analysis
- Customizable screening criteria
- Visual representation of momentum indicators

## Troubleshooting Tips
- **App won't deploy**: Ensure your repository has a main file that Streamlit can run.
- **Error messages**: Verify dependencies in your `requirements.txt` file.
- **Performance issues**: Optimize your data queries to minimize loading times.
